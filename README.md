## 4.安装webpack
安装：
webpack 是一个 npm 包，所以我们通过 npm 命令来全局安装：
npm install webpack -g
安装完成后，命令行下就有 webpack 命令可用，执行 webpack --help 可以查看 webpack 提供的各种命令。
